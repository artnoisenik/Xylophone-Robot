Instructions:

Our proprietary Sequence Generator allows you to hastly compose a song by typing in a series notes and chords

  • To add a note, simply type in its corresponding letter (i.e. a, b, c, d, e, f, or g)
  • To add a chord, add a ' character after the note's name (ex: e', b', g')
  • Seperate each note / chord with a space
  • The period character represents a rest
  • Each note / chord / rest will be treated as a quarter note
  • Example Song: Mary Had a Little Lamb
